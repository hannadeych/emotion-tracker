# Emotion Tracker

Простой браузерный трекер эмоций без фреймворков.

## Что делает
1. Пользователь выбирает эмоции (можно несколько)
2. Затем выбирает, где они ощущаются в теле
3. После этого получает варианты мягкой отработки состояния

Проект сделан как прототип / MVP для обсуждения логики и UX.

## Технологии
- HTML
- CSS
- Vanilla JavaScript
- Без фреймворков
- Работает прямо в браузере

## Как запустить
1. Открыть файл `index.html` в браузере
2. Ничего устанавливать не нужно

## Статус
MVP / прототип  
Планируется доработка логики рекомендаций и хранения данных.
